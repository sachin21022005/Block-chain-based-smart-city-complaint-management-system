# 🔗 Blockchain-Based Smart City Complaint Management System

## 📌 Description
A decentralized application that allows citizens to register and track complaints securely using blockchain technology, ensuring transparency and immutability.

## 🚀 Features
- Secure complaint registration
- Immutable complaint records
- Real-time complaint tracking
- Transparent system with no data tampering

## 🛠️ Tech Stack
- Ethereum Blockchain
- Solidity (Smart Contracts)
- HTML, CSS, JavaScript
- MongoDB

## ⚙️ How It Works
Complaints are stored on the blockchain using smart contracts. Once submitted, the data cannot be modified or deleted, ensuring transparency and trust.

## ▶️ How to Run
1. Install MetaMask or any Ethereum wallet
2. Deploy smart contract using Remix IDE
3. Connect frontend to blockchain
4. Run the application

## 🎯 Future Improvements
- Government dashboard integration
- Mobile application support
- Advanced analytics for complaints
